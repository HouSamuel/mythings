pip临时使用镜像站
```python
-i https://pypi.tuna.tsinghua.edu.cn/simple/
```
配置默认镜像站:
```bash
pip3 config set [--user] [--global] [--site] <section>.<key> <value>
```
- `--user`：将配置写入用户级配置文件（~/.config/pip/pip.conf 或 ~/.pip/pip.conf）
- `--global`：将配置写入系统级配置文件（/etc/pip.conf，需要管理员权限）
- `--site`：将配置写入当前虚拟环境的配置文件（如果虚拟环境已激活，则写入 $VIRTUAL_ENV/pip.conf）
- `global.index-url`：默认包索引 URL
- `global.trusted-host`：信任的主机（避免 SSL 警告）
##### 1.在此虚拟环境配置镜像站
```bash
pip3 config set --site global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/  &&  pip3 config set --site global.trusted-host https://pypi.tuna.tsinghua.edu.cn/simple/
```
##### 2.在用户级配置镜像站
```bash
pip3 config set --user global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/  &&  pip3 config set --user global.trusted-host https://pypi.tuna.tsinghua.edu.cn/simple/
```
##### 3.在系统级配置镜像站
```bash
sudo pip3 config set --global global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/  &&  sudo pip3 config set --global global.trusted-host https://pypi.tuna.tsinghua.edu.cn/simple/
```
##### 4.验证和删除
查看所有配置：
```bash
pip config list
```
删除配置
```bash
pip config unset global.index-url
```
## Macos
清除下载标签`sudo xattr -d com.apple.quarantine <your_app_path>`
```bash
sudo xattr -d com.apple.quarantine
```
homebrew苹果电脑安装脚本：
```
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```
homebrew苹果电脑卸载脚本：
```
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/HomebrewUninstall.sh)"
```
homebrew卸载所有已安装的软件
```bash
brew list --formula | xargs brew uninstall --ignore-dependencies  &&  brew list --cask | xargs brew uninstall --cask --force
```
